What is Carton?

Besides a pun with the lovely Cardboard from Google, Carton is a Do-It-Yourself SmartGlass (also called Optical Head Mounted Display, but it’s less sexy) using a mobile phone and simple materials like cardboard.

Why this project?

SmartGlasses are still expensive and hard to get, this project intends to fix theses issues. By allowing everyone to create some prototype, try new experiments and researches with, Carton aims to improve innovation and creativity by openly providing a simple way to access theses technologies.

How to get one?

Until someone sells a kit or even a fully created Carton, for now the easiest way to get one is to build it by yourself. It’s quick, affordable and easy. Just follow the construction guide. Everything is explained from the list of material and tools, to the template and each step.

Developer?

Carton for Android SDK enable you to easily create or adapts existing apps for Carton with auto-adapting screen and gesture recognition. It is also packed with the official demo app that you can explore for the sake of learning. It is fully compatible with augmented reality technologies (Vuforia, etc.).
